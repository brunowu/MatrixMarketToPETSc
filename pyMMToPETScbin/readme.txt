1. make sure you have installed python2.7 or plus

2. Execution: 
	python matrixConverter.py ${MMDIRECTORY}
	
	${MMDIRECTORY} means the location of matrix market file to be converted

	The converted petsc binary matrix will be in a new created file <converted>

	Attention: This converter convert all the matrix market files in the directory ${MMDIRECTORY}
